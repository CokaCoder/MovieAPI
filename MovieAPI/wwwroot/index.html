Jonathans Web API C# .NET Core

Jonathan Warner, Web API C# .NET Core 3.1

Please use Postman to test.

In the examples below, use the URI where "PORT" is your localhost port number, and "NUMBER" is a number between 1 and 10 to retrieve an existing id.

  • You can send a "GET" request to https://localhost:PORT/api/Movies - to retrieve a JSON payload containing the viewing statistics for all movies.
  • You can send a "GET" request to https://localhost:PORT/api/Movies/NUMBER - to retrieve a JSON payload containing all metadata for a specific movie.
  • You can send a "POST" request to https://localhost:51146/api/Movies with a payload simmilar to the example below, to "Create" a resource on the storage medium.


    • Example payload

    {
        "movieId": 99,
        "title": "The A Team",
        "averageWatchDurationS": "4309295",
        "watches": 1000,
        "releaseYear": "2001"
    }
